# KI-Nachrichtenregeln

Diese Datei beschreibt projektbezogene Regeln für KI-generierte Texte in diesem Repository.
Sie kann laufend erweitert werden.

## Aktuelle Regeln

1. Umlaute und Sonderzeichen im Deutschen werden immer korrekt dargestellt.
   Beispiele: ä, ö, ü, Ä, Ö, Ü, ß

2. Texte sind immer nutzerrelevant und motivieren zur passenden Aktion.

3. Texte vermeiden technische Implementierungsdetails, wenn sie für Nutzerinnen und Nutzer nicht relevant sind.


## Zweck

Diese Regeln gelten für Texte, die im Rahmen dieses Projekts von der KI formuliert oder überarbeitet werden.
